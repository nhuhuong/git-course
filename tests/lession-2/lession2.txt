1. git init: khởi tạo repo
2. git add <file name> : add file name vào vùng Staging
3. git add . : add tất cả file vào vung Staging 
4. git commit -m "message" : adđ file vào vung repository với note là message.

Bài 2 
1. tạo 3 file
2. git init: khởi tạo repo
3. git commit -m "init project": khoi tao voi message init project 
4. git add file 1 => add file 1 vao vung STaging 
5. git commit -m "add file 1" => add file 1 vao vung repo voi message "add file 1"
6. git add file -> loi, do khong co file nao ten file (3 file la file1, file2 va file3)
